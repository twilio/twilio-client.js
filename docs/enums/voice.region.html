Enumeration Region

Valid current regions.

deprecated

CLIENT-6831 This is no longer used or updated for checking validity of regions in the SDK. We now allow any string to be passed for region. Invalid regions won't be able to connect, and won't throw an exception.

CLIENT-7519 This is used again to temporarily convert edge values to regions as part of Phase 1 Regional. This is still considered deprecated.
